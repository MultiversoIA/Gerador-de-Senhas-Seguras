# 🔐 Gerador de Senhas Seguras

![Python Version](https://img.shields.io/badge/python-3.6%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Um utilitário de terminal robusto e seguro para a geração de senhas aleatórias. Criado para o projeto **MultiversoIA**.

## 🛡️ Segurança em Primeiro Lugar
Diferente de geradores comuns, este script utiliza o módulo **`secrets`** do Python, projetado especificamente para ser criptograficamente seguro. Isso garante que as senhas geradas não sejam previsíveis.

## ✨ Funcionalidades
- **Totalmente Customizável:** Escolha o tamanho (8 a 64 caracteres) e o tipo de caracteres.
- **Garantia de Diversidade:** O algoritmo garante que pelo menos um caractere de cada categoria escolhida estará presente na senha.
- **Interface Colorida:** Feedback visual claro no terminal para facilitar o uso.

## 🚀 Como Executar

1. Certifique-se de ter o **Python 3.6+** instalado.
2. Baixe o arquivo `main.py`.
3. No terminal, execute:
   ```bash
   python main.py
   ```

## 🛠️ Tecnologias Utilizadas
- **Python 3**
- Biblioteca `secrets` (Segurança Criptográfica)
- Biblioteca `string` (Manipulação de texto)
